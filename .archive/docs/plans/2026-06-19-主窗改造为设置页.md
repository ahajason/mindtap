# 主窗改造为设置页 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 把主窗从 V1.3 脚手架（顶栏 + 侧栏 + RecordTimeline + FAB）改成"中心按钮 + 单列滚动设置页"，同时落地 settings.json 持久化 + 日志基建 + 诊断面板。

**Architecture:** 三件事打包 — (1) `Settings` struct + Tauri State 缓存 + 4 个 side-effect appliers + emit `settings-changed` 事件；(2) `log` + `env_logger` 三档输出（stderr / 文件 / 200 条 ring）；(3) Diagnostics 面板拉 hotkey/active task/DB/ax 状态。UI 用 Glassic UI（shadcn 源码复制）16 组件 + Sonner toast。Vivid 色散、跨平台 hotkey 迁移、NSVisualEffectView 等留 v1.1+。

**Tech Stack:** Tauri 2 + React 19 + TypeScript + Vite 7 + Tailwind 3 + Radix + sonner + vitest (dev) + log/env_logger (Rust) + cocoa/objc (macOS only).

### Task 2.2: 写 `LogEntry` + `RingBuffer`

- [ ] **Step 1: 写 `src-tauri/src/log/ring.rs`**

完整内容：
```rust
use serde::Serialize;
use std::collections::VecDeque;
use std::sync::{Arc, Mutex};

#[derive(Debug, Clone, Serialize)]
#[serde(rename_all = "camelCase")]
pub struct LogEntry {
    pub ts: i64,
    pub level: String,
    pub target: String,
    pub message: String,
}

#[derive(Debug)]
pub struct RingBuffer {
    inner: Mutex<VecDeque<LogEntry>>,
    cap: usize,
}

impl RingBuffer {
    pub fn new(cap: usize) -> Self {
        Self { inner: Mutex::new(VecDeque::with_capacity(cap)), cap }
    }
    pub fn push(&self, e: LogEntry) {
        let mut g = self.inner.lock().unwrap();
        if g.len() == self.cap { g.pop_front(); }
        g.push_back(e);
    }
    pub fn recent(&self, n: usize) -> Vec<LogEntry> {
        let g = self.inner.lock().unwrap();
        g.iter().rev().take(n).cloned().collect::<Vec<_>>().into_iter().rev().collect()
    }
    pub fn shared(self) -> Arc<Self> { Arc::new(self) }
}

pub type SharedRing = Arc<RingBuffer>;
```

### Task 2.3: 写 `log::init` + file writer + rotation

- [ ] **Step 1: 写 `src-tauri/src/log/mod.rs`**

```rust
use crate::log::ring::{LogEntry, RingBuffer, SharedRing};
use std::fs::{self, File, OpenOptions};
use std::io::Write;
use std::path::PathBuf;
use std::sync::{Arc, Mutex, OnceLock};
use tauri::Manager;

pub mod ring;

const RING_CAP_DEFAULT: usize = 200;
const FILE_MAX_BYTES: u64 = 5 * 1024 * 1024; // 5MB

static RING: OnceLock<SharedRing> = OnceLock::new();

pub fn shared_ring() -> SharedRing { RING.get().cloned().expect("log::init not called") }

pub fn init(app: &tauri::AppHandle) -> Result<(), String> {
    let ring = Arc::new(RingBuffer::new(RING_CAP_DEFAULT));
    let _ = RING.set(ring.clone());

    let log_path = log_path(app)?;
    rotate_if_needed(&log_path).map_err(|e| e.to_string())?;
    let file = open_log_file(&log_path).map_err(|e| e.to_string())?;
    let file = Arc::new(Mutex::new(file));

    let writer = DualWriter { file, ring };

    env_logger::Builder::from_env(env_logger::Env::default().default_filter_or("info"))
        .target(env_logger::Target::Stderr)
        .format(move |buf, record| {
            let ts = chrono_now_ms();
            let entry = LogEntry {
                ts, level: record.level().to_string().to_lowercase(),
                target: record.target().to_string(), message: format!("{}", record.args()),
            };
            // Mirror to ring + file
            let ring = writer.ring.clone();
            ring.push(entry.clone());
            if let Ok(mut f) = writer.file.lock() {
                let _ = writeln!(f, "{} [{}] {}: {}", entry.ts, entry.level, entry.target, entry.message);
                let _ = f.flush();
            }
            // Stderr output
            writeln!(buf, "{} [{}] {}: {}", entry.ts, entry.level, entry.target, entry.message)
        })
        .init();

    log::info!("log::init complete; path={:?}", log_path);
    Ok(())
}

pub fn apply_logging(level_filter: &str) {
    // Hot-update env_logger filter; we re-set the RUST_LOG via env
    std::env::set_var("RUST_LOG", level_filter);
    // env_logger reads at init only, so for v1 we log the change and let next restart pick up.
    log::info!("logging.level change requested: {level_filter} (effective after restart)");
}

fn log_path(app: &tauri::AppHandle) -> Result<PathBuf, String> {
    let dir = app.path().app_log_dir().map_err(|e| e.to_string())?;
    fs::create_dir_all(&dir).map_err(|e| e.to_string())?;
    Ok(dir.join("mindtap.log"))
}

fn open_log_file(path: &PathBuf) -> std::io::Result<File> {
    OpenOptions::new().create(true).append(true).open(path)
}

fn rotate_if_needed(path: &PathBuf) -> std::io::Result<()> {
    let Ok(meta) = fs::metadata(path) else { return Ok(()) };
    if meta.len() < FILE_MAX_BYTES { return Ok(()) }
    let old = path.with_extension("log.old");
    let _ = fs::remove_file(&old);
    fs::rename(path, old)
}

fn chrono_now_ms() -> i64 {
    use std::time::{SystemTime, UNIX_EPOCH};
    SystemTime::now().duration_since(UNIX_EPOCH).map(|d| d.as_millis() as i64).unwrap_or(0)
}

struct DualWriter { file: Arc<Mutex<File>>, ring: SharedRing }
```

use crate::settings::schema::Settings;
use std::fs;
use std::io::Write;
use std::path::PathBuf;
use tauri::{AppHandle, Manager};

pub mod cmd;
pub mod schema;

pub const CURRENT_VERSION: u32 = 1;
pub const SETTINGS_FILENAME: &str = "settings.json";

pub fn settings_path(app: &AppHandle) -> Result<PathBuf, String> {
    let dir = app.path().app_config_dir().map_err(|e| e.to_string())?;
    fs::create_dir_all(&dir).map_err(|e| e.to_string())?;
    Ok(dir.join(SETTINGS_FILENAME))
}

pub fn load_from_file(path: &PathBuf) -> Settings {
    let raw = match fs::read_to_string(path) {
        Ok(s) => s,
        Err(_) => return Settings::default(),
    };
    let v: serde_json::Value = serde_json::from_str(&raw).unwrap_or_else(|_| {
        log::warn!("settings.json parse failed; using defaults");
        serde_json::Value::Null
    });
    let version = v.get("version").and_then(|x| x.as_u64()).unwrap_or(0);
    match version {
        0 => migrate_v0(v),  // greenfield: 0 → 1
        1 => serde_json::from_value(v).unwrap_or_default(),
        v if v > CURRENT_VERSION => {
            log::warn!("settings.json version {v} newer than app; using defaults");
            Settings::default()
        }
        _ => Settings::default(),
    }
}

fn migrate_v0(v: serde_json::Value) -> Settings {
    // v1 spec: greenfield migration. Future v0→v2 will go here.
    let _ = v;
    Settings::default()
}

pub fn load_or_default(app: &AppHandle) -> Settings {
    let path = match settings_path(app) {
        Ok(p) => p,
        Err(_) => return Settings::default(),
    };
    let mut s = load_from_file(&path);
    if s.meta.first_launch_at == 0 {
        s.meta.first_launch_at = now_ms();
    }
    s.meta.total_launches = s.meta.total_launches.saturating_add(1);
    let _ = save(app, &s);  // best-effort
    s
}

pub fn save(app: &AppHandle, s: &Settings) -> Result<(), String> {
    let final_path = settings_path(app)?;
    let tmp_path = final_path.with_extension("json.tmp");
    let json = serde_json::to_string_pretty(s).map_err(|e| e.to_string())?;
    {
        let mut f = fs::File::create(&tmp_path).map_err(|e| e.to_string())?;
        f.write_all(json.as_bytes()).map_err(|e| e.to_string())?;
        f.sync_all().map_err(|e| e.to_string())?;
    }
    fs::rename(&tmp_path, &final_path).map_err(|e| e.to_string())?;
    Ok(())
}

fn now_ms() -> i64 {
    use std::time::{SystemTime, UNIX_EPOCH};
    SystemTime::now().duration_since(UNIX_EPOCH).map(|d| d.as_millis() as i64).unwrap_or(0)
}

```rust
use crate::settings::schema::{FloatingSettings, HotkeySettings, LoggingSettings};
use tauri::{AppHandle, Manager, PhysicalSize};
use tauri_plugin_global_shortcut::{Code, GlobalShortcutExt, Modifiers, Shortcut, ShortcutState};

pub fn apply_hotkey(app: &AppHandle, hotkey: &HotkeySettings) -> Result<(), String> {
    let gs = app.global_shortcut();
    // Unregister all existing first (idempotent; tauri will silently ignore unknown)
    let Ok(current) = current_shortcut(app) else { return Ok(()); };
    if let Ok(cur) = current { let _ = gs.unregister(cur); }
    let new = hotkey.to_shortcut().map_err(|e| e.to_string())?;
    gs.on_shortcut(new, |_app, _sc, _event| { /* placeholder, real handler in lib.rs */ })
        .map_err(|e| e.to_string())?;
    log::info!("hotkey registered: {hotkey:?}");
    Ok(())
}

fn current_shortcut(app: &AppHandle) -> Result<Option<Shortcut>, String> {
    let st = app.state::<crate::settings::SettingsState>();
    let cur = st.0.lock().unwrap().hotkey.to_shortcut().ok();
    Ok(cur)
}

pub fn apply_autostart(app: &AppHandle, enable: bool) -> Result<(), String> {
    let al = app.autolaunch();
    let on = al.is_enabled().unwrap_or(false);
    if enable && !on { al.enable().map_err(|e| e.to_string())?; log::info!("autostart enabled"); }
    else if !enable && on { al.disable().map_err(|e| e.to_string())?; log::info!("autostart disabled"); }
    Ok(())
}

pub fn apply_floating_geometry(app: &AppHandle, f: &FloatingSettings) -> Result<(), String> {
    let w = app.get_webview_window("floating").ok_or("floating window missing")?;
    w.set_min_size(Some(PhysicalSize::new(f.fold_w as u32, f.fold_h as u32))).map_err(|e| e.to_string())?;
    w.set_max_size(Some(PhysicalSize::new(f.max_w as u32, f.max_h as u32))).map_err(|e| e.to_string())?;
    log::info!("floating geometry applied: min={}x{}, max={}x{}", f.fold_w, f.fold_h, f.max_w, f.max_h);
    Ok(())
}

pub fn apply_logging(l: &LoggingSettings) {
    let level = match l.level {
        crate::settings::schema::LogLevel::Error => "error",
        crate::settings::schema::LogLevel::Warn => "warn",
        crate::settings::schema::LogLevel::Info => "info",
        crate::settings::schema::LogLevel::Debug => "debug",
        crate::settings::schema::LogLevel::Trace => "trace",
    };
    crate::log::apply_logging(level);
}

import { useCallback, useEffect, useRef, useState } from 'react'
import { events, settings } from '@/lib/tauri-bridge'
import { log } from '@/lib/log'
import type { Settings } from '@/settings/schema'

export type SetStatus = 'idle' | 'saving' | 'error'

export function useSettings() {
  const [data, setData] = useState<Settings | null>(null)
  const [status, setStatus] = useState<SetStatus>('idle')
  const pending = useRef<Partial<Settings> | null>(null)
  const timer = useRef<number | null>(null)

  useEffect(() => {
    settings.get().then(setData).catch(e => { log.error('settings_get failed', e); setStatus('error') })
    const un = events.settingsChanged<Settings>(s => setData(s))
    return () => { un.then(u => u()) }
  }, [])

  const flush = useCallback(async () => {
    if (!pending.current) return
    const next = pending.current
    pending.current = null
    setStatus('saving')
    try {
      const merged = { ...(data ?? {} as Settings), ...next } as Settings
      const saved = await settings.set(merged)
      setData(saved); setStatus('idle')
    } catch (e) {
      log.error('settings_set failed', e); setStatus('error')
      // refresh to revert UI
      settings.get().then(setData).catch(() => {})
    }
  }, [data])

  const setField = useCallback(<K extends keyof Settings>(key: K, value: Settings[K]) => {
    setData(prev => prev ? { ...prev, [key]: value } : prev)
    pending.current = { ...(pending.current ?? {}), [key]: value }
    if (timer.current) window.clearTimeout(timer.current)
    timer.current = window.setTimeout(() => { flush() }, 300)
  }, [flush])

  const setSettings = useCallback((next: Settings) => {
    setData(next); pending.current = next
    if (timer.current) window.clearTimeout(timer.current)
    flush()
  }, [flush])

  const reset = useCallback(async () => {
    setStatus('saving')
    try { const r = await settings.reset(); setData(r); setStatus('idle') }
    catch (e) { log.error('settings_reset failed', e); setStatus('error') }
  }, [])

  return { settings: data, setField, setSettings, reset, status }
}
```

export function useAccessibility() {
  const [status, setStatus] = useState<boolean | null>(null)
  const refresh = useCallback(() => { accessibility.status().then(setStatus).catch(() => setStatus(null)) }, [])

  useEffect(() => {
    refresh()
    const id = window.setInterval(refresh, 5000)
    const onFocus = () => refresh()
    window.addEventListener('focus', onFocus)
    return () => { window.clearInterval(id); window.removeEventListener('focus', onFocus) }
  }, [refresh])

  return { status, request: accessibility.requestPrompt, openSettings: accessibility.openSettings, refresh }
}

## PR 9: KeyRecorder component + useKeyRecorder hook

**Files:**
- Create: `src/hooks/useKeyRecorder.ts`, `src/settings/components/KeyRecorder.tsx`
- Test: `useKeyRecorder.test.ts`, `KeyRecorder.test.tsx`

**Interfaces:**
- `useKeyRecorder({ value, onChange })` returns `{ state: 'idle'|'recording'|'saving'|'cancel', start, confirm, cancel, display }`
- 4-phase FSM:
  - `idle` → `start()` → `recording`
  - `recording` → `keydown(valid)` → `saving` (调 onChange) → `idle`
  - `recording` → `keydown(Esc)` → `cancel` → `idle`
  - `recording` → `cancel()` → `cancel` → `idle`

### Task 9.1: 写 `useKeyRecorder` + 测试

- [ ] **Step 1: 写 `src/hooks/useKeyRecorder.ts`**

```ts
import { useCallback, useEffect, useState } from 'react'

export interface KeyCombo { modifiers: number; code: string }
export type RecorderState = 'idle' | 'recording' | 'saving' | 'cancel'

const MOD_ORDER = ['Meta', 'Control', 'Alt', 'Shift']  // display order
const MOD_BITS: Record<string, number> = { Shift: 1, Control: 2, Alt: 4, Meta: 8 }

function isModifierOnly(code: string) {
  return ['ShiftLeft', 'ShiftRight', 'ControlLeft', 'ControlRight', 'AltLeft', 'AltRight', 'MetaLeft', 'MetaRight'].includes(code)
}

function formatKey(c: KeyCombo) {
  const mods = MOD_ORDER.filter(m => c.modifiers & MOD_BITS[m]).join('+')
  return mods ? `${mods}+${c.code}` : c.code
}

export function useKeyRecorder({ value, onChange }: { value: KeyCombo; onChange: (v: KeyCombo) => void }) {
  const [state, setState] = useState<RecorderState>('idle')
  const [draft, setDraft] = useState<KeyCombo | null>(null)

  const start = useCallback(() => { setState('recording'); setDraft(null) }, [])
  const cancel = useCallback(() => { setState('cancel'); setDraft(null); setTimeout(() => setState('idle'), 0) }, [])

  useEffect(() => {
    if (state !== 'recording') return
    const handler = (e: KeyboardEvent) => {
      if (e.key === 'Escape') { e.preventDefault(); cancel(); return }
      if (isModifierOnly(e.code)) return
      // require at least one modifier
      const mods = (e.shiftKey ? 1 : 0) | (e.ctrlKey ? 2 : 0) | (e.altKey ? 4 : 0) | (e.metaKey ? 8 : 0)
      if (mods === 0) return
      // disallow single-letter without modifier — already enforced above
      e.preventDefault()
      const next: KeyCombo = { modifiers: mods, code: e.code }
      setDraft(next); setState('saving')
      try { onChange(next) } finally { setTimeout(() => setState('idle'), 0) }
    }
    window.addEventListener('keydown', handler)
    return () => window.removeEventListener('keydown', handler)
  }, [state, onChange, cancel])

  return { state, start, cancel, draft, display: formatKey(draft ?? value) }
}
```

#[tauri::command]
pub fn reset_window_positions(app: AppHandle) -> Result<(), String> {
    for label in ["main", "floating"] {
        if let Some(w) = app.get_webview_window(label) {
            let _ = w.center();
            let _ = w.unmaximize();
        }
    }
    // also clear windowState in settings
    let st = app.state::<crate::settings::SettingsState>();
    let mut s = st.0.lock().unwrap();
    s.window_state.main = None;
    s.window_state.floating = None;
    let _ = crate::settings::save(&app, &s);
    let _ = tauri::Emitter::emit(&app, "settings-changed", &*s);
    Ok(())
}
