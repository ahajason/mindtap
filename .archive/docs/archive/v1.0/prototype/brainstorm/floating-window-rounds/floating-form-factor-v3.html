Pros

  • 折叠最紧凑（40px），不打扰

  • Mac/Windows Web 一致

Cons

  • 展开后宽度大（420px）覆盖顶部空间
  • 跟全屏 App 顶栏冲突
  • 下拉动画比左右展开更"打扰"
