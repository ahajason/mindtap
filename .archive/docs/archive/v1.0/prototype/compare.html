A
Minimal 极简
Q1-light-creative.html · 1,604 行 · Subtle 4 档 · Desktop
核心定位
玻璃存在感最小化，让内容做主角
采用 Subtle 4 档（ultraThin / thin / regular / thick，blur 10/14/16/18px），并用 grid-break（main margin-left: -12px 与 240px 侧栏重叠 12px）打破栅格边界 — 这是 8 视角中唯一打破栅格的实现。
原则体现
  • Content First ★★★★★ — 唯一让 stats/records 内容层也带轻玻璃
  • Adaptive Material ★★★★★ — 显式 4 档玻璃对应 HIG Materials 4-tier
  • Readability ★★★★★ — stat__value 28px 保证可读
  • Functional Design ★★★ — FAB 拖拽 + 位置记忆 + Esc/N
特色属性
  • Subtle 4 档（10/14/16/18px）
  • Desktop（240px 侧栏 + 1fr）
  • grid-break（main 与侧栏重叠 12px）
  • 卡片倾斜 ±0.4°-1° · 中低密度 · 单 light 主题
独特价值
唯一采用 grid-break（内容区 margin-left: -12px 与 240px 侧栏重叠 12px）打破栅格边界的视角。其它视角都遵循严格的 12-column 栅格，A 是 8 视角中"反栅格"的唯一证明。
B
Navigation First 导航优先
Q2-light-strict.html · 1,182 行 · Subtle 单档 · Desktop · macOS 风
核心定位
玻璃严守 4 个功能层，内容层完全无玻璃
最克制实现：玻璃仅用于 menubar / sidebar / FAB / modal / bottom-tab 5 个功能层。内容层（stats / records / review）完全无 backdrop-filter，让数据"穿透"而出。
原则体现
  • Content First ★★★★★ — 内容层零 backdrop-filter
  • Nav Above Content ★★★★★ — menubar+sidebar+bottom-tab 三层 sticky
  • Clear Hierarchy ★★★★★ — 三层导航严格 z-index
  • Readability ★★★★★ — 唯一把 menubar 压到 32px
特色属性
  • Subtle 单档 glass-regular（blur 24px saturate 180%）
  • Desktop（macOS 风 side+top 双 sticky）
  • 4 色 mesh 背景（粉/橙/紫/蓝 4 点 radial-gradient）
  • 中密度 · 单 light 主题
独特价值
唯一内容层完全无任何玻璃（无 backdrop-filter、无半透明白底）的视角。其它视角至少在某些区块会带轻玻璃，B 是 8 视角中"功能/内容"分层最严格的证明。
C
Spatial Depth 空间层级
Q4-heavy-strict.html · 1,555 行 · Rich · Desktop
核心定位
Rich 玻璃 + 5 层阴影构建"内容 vs 功能"双层空间
采用 Rich 强度（blur 40px + saturate 220% + 5 层阴影 + noise 0.07），对应 HIG `Material.thick`。仅在功能层（menubar/sidebar/FAB/modal）应用玻璃，内容层保持纯净。
原则体现
  • Content First ★★★★★ — 内容层零玻璃
  • Clear Hierarchy ★★★★★ — 5 层阴影 stack 专属功能层
  • Readability ★★★★★ — thick 档保证 28px 统计数字 100% 可读
  • Functional Design ★★★★★ — scroll edge effect + FAB breath + settle 落定
特色属性
  • Rich blur 40px + saturate 220% + 5 层阴影 + noise 0.07
  • Desktop · 唯一 --mx/--my CSS 变量 + JS 实时同步鼠标跟随 specular
  • 中密度 · 单 light 主题
独特价值
唯一实现 settle 入场动画（先弹起 translateY(-4px) → 落下 translateY(2px) → 稳定 0）。同时是 8 视角中阴影层数最多（5 层）的视角，提供最明确的"功能浮于内容之上"空间感。
D
Immersive 沉浸
D5-immersive.html · 2,064 行 · Rich · macOS · 18 事件
核心定位
背景延伸 + 玻璃流动的边缘到边缘沉浸视角
实现 Apple HIG Background Extension Effect — 背景跨越状态栏与玻璃区。6 层材质系统（L1 Base / L2 Refraction / L3 Specular / L4 Dynamic Tint / L5 Noise / L6 Shadow）配合 mouse-following tint。
原则体现
  • Content First ★★★★ — 8 records + 4 stat + 3 词复盘无玻璃
  • Adaptive Material ★★★★★ — 6 层材质系统完整
  • Functional Design ★★★★★ — 18 JS 事件 + FAB 拖动阈值 + 位置记忆 + modal 锚定
  • Consistency ★★★★ — 双轨 light/dark
特色属性
  • Rich `.glass` blur 32px/saturate 200%；`.glass-thick` blur 48px/saturate 220%
  • 跨端响应式（grid 280px sidebar + 1fr desktop；移动端 1fr + 68px sidebar + 64px pill bottom-tab）
  • 5 层 mesh + body::before drift 18s + body::after 顶部光晕 12s
  • 中低密度 · light + dark 双轨
独特价值
唯一 modal 开启动画以 FAB 实时坐标为 transform-origin 锚定生长（SwiftUI `.confirmationDialog` 等价）。其它视角的 modal 都是从屏幕中心或顶部弹出，D 是唯一"从用户最近一次操作点"生长的视角。
E
Productive 生产力
D6-productive.html · 1,713 行 · Subtle · Desktop · 最高密度
核心定位
紧凑高密度工作区，6 stat 网格 + 24 行真实表格 + 24h 时间轴
FE-2.38 修复后采用真实 Subtle Glass（blur 18-24px / saturate 150-180%）。FE-2.38 之前是假玻璃，修复后符合 HIG Material.thin 的物理参数。
原则体现
  • Content First ★★★★★ — 内容层透明无玻璃；同行视图可读 30+ 字段
  • Clear Hierarchy ★★★★★ — 6 列 stats > 24h 时间轴 > 24 行记录表 > 复盘
  • Functional Design ★★★★★ — 顶部工具条 HIG 3 区 + 搜索 + 3 filter chips
  • Readability ★★★★★ — 14px 行高，30+ 字段不挤压
特色属性
  • Subtle（blur 18-24px / saturate 150-180%）
  • Desktop 为主（768px 切 50px bottom-tab）
  • 4 色 accent mesh 背景（5 点 radial）
  • 最高密度（6 stat + 24 行真实 <table> + 9 modal 按钮 + 50px tab）
  • 单一暖灰 #F5F4F0 + 4 色 brand mesh
独特价值
唯一拥有 24h timeline strip（仿 slider 含 now 指示器 thumb）的视角。其它视角用 SVG/网格呈现一天，E 是唯一有"时间刻度 + now 标记"的实现，强调生产力场景的时间感知。
F
Experimental 实验
Q3-heavy-creative.html · 1,555 行 · Experimental · Desktop
核心定位
5 层堆叠 + shimmer + pulse-glow 的创意边界探索器
有意识违反 HIG — 内容层 100% 全玻璃覆盖（包括 stats/records），但用 5 层阴影（base + 折射 + 边缘 + 阴影 + 噪点 + 鼠标 specular）保证可读性。6 个 @keyframes 全部无限循环。
原则体现
  • Adaptive Material ★★★★ — Rich 强度 + 内容层 100% 全玻璃
  • Functional Design ★★★★ — FAB hover 触发 pulse-glow 无限循环 + shimmer sheen
  • Clear Hierarchy ★★★ — 内容层与功能层区分弱
  • Readability ★★ — 内容层带玻璃后可读性下降
特色属性
  • Experimental blur 40-48px / saturate 220% / 5 层阴影叠加 / noise overlay
  • Desktop 优先（移动端 bottom-tab 降级）
  • pulse-glow + shimmer 双无限动画
  • 4 色 stat tint + 9 色 record icon 渐变 · 中低密度 · 单 light
独特价值
唯一把"无限循环动画 + sheen 扫光"作为 FAB 主操作反馈的视角。其它视角的 FAB 用静态或单次动画反馈，F 是 8 视角中"反 HIG 但有趣"的实验场。
G
Adaptive 跨端自适应
D7-adaptive.html · 2,268 行 · 3 档动态 · Mobile + Windows + macOS · 29 事件
核心定位
同一份 DOM、3 平台切换、4 季主题的跨端自适应验证原型
3 套独立 CSS 变量驱动玻璃强度（mobile blur 20 / win 26 / macos 36）。29 个 addEventListener 中包含平台切换 / 主题切换 / 季节切换 / 折叠 / modal / FAB / 搜索 / ⌘K 快捷键。
原则体现
  • Nav Above Content ★★★★★ — 3 套最优导航：mobile bottom-tab · Windows sidebar+header · macOS menubar+sidebar+Inspector
  • Adaptive Material ★★★★★ — 3 套独立 CSS 变量
  • Consistency ★★★★★ — 共享 DOM + 共享品牌调色板
  • Functional Design ★★★★★ — 29 事件最完整
特色属性
  • 3 档动态（mobile blur 20px / windows 26px / macOS 36px）
  • Mobile + Windows + macOS（带 menubar + Inspector 三栏 NavigationSplitView）
  • 中创意（平台切换本身即创意）· 中密度
  • 4 季（淡雅化 春 #F9A8D4 / 夏 #FB923C / 秋 #D97706 / 冬 #A5B4FC）+ 4 色 brand 双轨
独特价值
唯一实现 3 平台完整切换 + 4 季主题跨平台运行，是 mindtap 跨端一致性的"活体验证机"。其它视角只关注单一平台，G 是 8 视角中"一份代码，三种体验"的唯一证明。
