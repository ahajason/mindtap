A

顶部胶囊条（最紧凑）

横向 320×40px，紧贴屏幕顶部居中。Liquid Glass 长条，左侧"当前任务" + 计时器，右侧 3 个图标按钮（待办/灵感/快打卡）。点击输入区在窗口下方向下滑出。

⭐ 我的推荐：方案 B（矩形极简卡）

理由：① 折叠态信息密度刚好（任务+计时+3类型 Tab），工作中一眼看到当前任务，解决"遗忘"痛点；② 展开态不挤占主应用视野；③ Mac + Windows 实现一致（Web 全做），不依赖 R1 高风险项；④ 3 类型 Tab 比 3 个图标按钮可发现性强，更符合 0 思考成本。

风险：96px 高度对极小屏笔记本偏大 → 可加"超紧凑模式"（折叠态压缩到 64px，仅显示任务名+计时）。
