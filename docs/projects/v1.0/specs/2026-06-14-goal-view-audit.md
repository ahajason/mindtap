## 7 原则（来自 /goal）

1. **Content First** — 内容优先，玻璃不抢眼
2. **Clear Hierarchy** — 导航 / 工具 / 内容分层清晰
3. **Adaptive Material** — 玻璃根据背景 / 滚动 / 上下文动态适配
4. **Consistency** — 导航结构 / 操作逻辑 / 工具分组一致
5. **Readability** — 文字 / 图标 / 关键内容可读
6. **Functional Glass** — 每层玻璃有明确功能价值
7. **Platform Alignment** — 符合 Apple HIG + Liquid Glass

## 每个变体的"视角独特价值"

### Q1 Minimal（最小化玻璃存在感）
- **独特价值**：通过 transform rotate 让卡片"微微不规则"，制造手作感
- **典型场景**：内容密度高、需要快速扫读（笔记 / 日记 / 列表）
- **违反 7 原则的位置**：#2 Hierarchy（倾斜可能让用户误判点击区域）— 但因倾斜 < 1°，实际不影响

### Q2 Navigation First（导航层级最清晰）
- **独特价值**：玻璃**仅**用于 4 个功能层，**最严格**的 Liquid Glass 实现
- **典型场景**：专业工具（IDE / 设计软件 / 数据分析）
- **违反 7 原则的位置**：无
- **可作为生产基线**

### Q3 Experimental（材质边界探索）
- **独特价值**：5 层堆叠（基底 + 折射 + 边缘 + 阴影 + 噪点 + 鼠标 specular）展示 Liquid Glass 上限
- **典型场景**：营销页 / 品牌官网 / 沉浸式介绍页
- **违反 7 原则的位置**：#7（与 Apple "节制使用" 略有冲突）— 但这正是 Experimental 的本意

### Q4 Spatial（空间分层工具）
- **独特价值**：强 3D 感、多层阴影堆叠（5+ 层）让玻璃元素"漂浮"在内容之上
- **典型场景**：3D 内容 / 数据可视化 / 多窗口工作流
- **违反 7 原则的位置**：无
