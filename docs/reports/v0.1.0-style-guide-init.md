## 下一阶段 (V0.2) 待办建议

按 V0.1.0 业务范围文档 `c6382ba` 的规划,V0.2 起步应解决以下:

| 优先级 | 主题 | 范围 | 备注 |
|---|---|---|---|
| P0 | 浮窗 (Floating Window) | Tauri 独立窗口 + 全局快捷键 + Liquid Glass surface | 用户已在 settings 设计文档 (2026-06-19) 中明确要求 |
| P0 | 主窗口 → 设置入口 | 设置面板 / 持久化 (settings.json) | 同上 spec |
| P1 | 本地 SQLite 接入 | rusqlite / sqlx + 实体层 + migration | V0.1 完全是 UI 骨架,无数据层 |
| P1 | 浮窗日志环 (log ring) | 滚动 buffer + 简单 dump | 在 settings 设计中提到 |
| P2 | Style guide 视觉回归测试 | Playwright / snapshot | 当前 11 ui 组件仅 smoke test,无视觉断言 |
| P2 | i18n 基础设施 | `react-i18next` + 简中/英文 key 文件 | V0.1 完全中文 |
| P3 | Sonner 包装层升级 | 多 variant / action button / promise API | 当前 Toaster 只接 `richColors`,未暴露 variant |
