## Cursor 审计落地

| 元素 | cursor 类 | 状态 |
|---|---|---|
| input | `cursor-text` + `disabled:cursor-not-allowed disabled:opacity-50` + `read-only:cursor-text` | ✅ |
| textarea | 同 input | ✅ |
| button | `disabled:cursor-not-allowed`(V0.1.2 已做) | ✅ |
| NavLink / Tabs Trigger | `pointer`(元素语义自带) | ✅ |
| Label / Badge / Card / Separator / Tooltip / Dialog / Sidebar aside | default(元素语义/装饰元素) | ✅ |

**铁律**:`cursor-text` 仅用于 input/textarea(文字可编辑/可选);静态文字/容器/装饰元素一律 default;disabled 态全 UI 库 `cursor-not-allowed` 一致。
