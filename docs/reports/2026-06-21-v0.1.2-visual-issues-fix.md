## 推迟到 V0.1.3+ 的事项

| 项 | 原因 | 影响 |
|---|---|---|
| **Dark mode** | Tailwind 4 `@theme {}` 静态,需 theme provider + token doubling(3-5 天工作量) | 显式 deferred 到 V0.1.3+ |
| **NSVisualEffectView Tier 2** | Rust 端 NSException 风险,需独立测试 | V0.2.x |
| **Tab bar spec 实现** | 暂无 Tab bar 组件,需先有组件才需要 spec | V0.1.3+ 视需求 |
| **Toolbar spec 实现** | 同上 | — |
| **Shadow blue tint / Sidebar divider / Input focus 弱等 P2 视觉细节** | 本计划优先级未覆盖,未影响 WCAG 合规 | V0.1.3+ |
| **Glass not batched performance** | 需重构 layout 结构,影响大 | V0.2.x |
| **PageHeader / Section 各自 active 视觉统一** | 本计划只做 Sidebar NavLink;PageHeader/Section active 延后 | V0.1.3+ |
| **Input / Textarea / CodeBlock 玻璃滥用** | Task 4 已通过子页面实色间接修复 | V0.1.3+ |

```
玻璃不是装饰,是结构。   ← P0-2 修复,内容层不再滥用
可读性不是偏好,是底线。  ← P1-spacing 修复 + Reduce Transparency 降级
一致性不是重复,是约定。  ← Focus 3 层模型 + Card padding prop
```
