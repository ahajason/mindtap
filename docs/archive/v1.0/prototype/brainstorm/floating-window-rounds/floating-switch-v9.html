📐 4 条产品逻辑（v8 → v9 改变）

① 切换 = 从"未开始+已暂停"清单选 1 个：展开态有独立"切换"按钮，点击展开清单（5-10 项），点击即成为 Focus。类似 macOS 控制中心切换音源。
② 暂停 = 主按钮：暂停后还在进行中（任务保留），视觉权重高 → 蓝色高亮实心填充。完成 = 次要（任务消失） → 淡绿低饱和描边。
③ 按钮 = "开始计时 ⏵"：去掉"保存并开始计时 ⏵"长文本，改为"开始计时 ⏵"。新建 = 默认就开始计时（合并动作）。
④ 移除"保存即开始"提示：铁律"0 思考成本"——按钮文字已经表达，不需要额外解释。
