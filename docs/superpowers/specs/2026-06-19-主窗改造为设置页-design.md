# 主窗改造为设置页 — 2026-06-19

## 目标

把当前主窗的 V1.3 脚手架（顶栏 + 侧栏 + RecordTimeline + FAB）替换成"中心按钮 + 单列滚动设置页"，同时落地三件事：

1. **设置持久化** —— 把代码里隐含可配但无 UI 暴露的项（autostart、快捷键、浮窗几何、外观等）落到 settings.json，UI 完整编辑。
2. **日志基建** —— `log` + `env_logger` → stderr + 文件 + 200 条内存 ring，给"为什么浮窗不出现"这类问题留下诊断通道。
3. **诊断面板** —— 辅助功能授权 + 快捷键注册结果 + DB 状态 + ring viewer，主窗内一键查当前阻塞点。

## 现状

**主窗内容**（`src/App.tsx`）：顶栏 brand + 侧栏(视图/浮动窗/版本号) + RecordTimeline + FAB（"+"）。

**隐含可配但无 UI**：autostart（已通过托盘可切）、全局快捷键（`lib.rs` 硬编码 `SUPER|SHIFT|Space`）、浮窗几何（3 处硬编码）、hover-out 延迟、动画时长、macOS panel flag。

**隐含可观测但无 UI**：autostart enabled、快捷键注册结果、active task、pending idea、今日 check-in、DB 路径/大小、macOS Accessibility 授权、App 版本。

**日志现状**：Rust 只有 `tray/menu.rs:75` 一行 `eprintln!`，无 `log`/`tracing`，无 init；React 只有 `floating/App.tsx` 的 `DEBUG=true` dlog。

**视觉规范**：以 [`docs/projects/design-system/glassmorphism-impl-spec.md`](../projects/design-system/glassmorphism-impl-spec.md) 为强制项（已 `established: 2026-06-19, status: active`），**冷蓝浅玻璃拟态**，三 tier token（L1 blur(20px) / L2 blur(24px) / L3 blur(28px)）+ 蓝调弥散阴影 `rgba(0,30,80, 0.08~0.12)`。

## 设计决策

| # | 决策 | 原因 |
|---|---|---|
| 1 | **Glassic UI**（shadcn 源码复制）+ Tailwind + Radix 原语 | 源码入仓可全量改写；Tailwind 配置 spec tokens 一次性注入；15 个组件覆盖设置页 80% 控件；不绑开发逻辑 |
| 2 | 不引 `tauri-plugin-store` / `tailwindcss-animate` 之外的 CSS 框架 | 直接 serde_json + atomic write 更轻；plugin-store API 厚度无收益 |
| 3 | settings 存 `app_config_dir()/settings.json`，**不**塞 SQLite | 不污染业务表；JSON 直观；规模 < 1KB；atomic rename 易实现 |
| 4 | 单文件 settings.json 包含 `meta` / `startup` / `hotkey` / `floating` / `windowState` / `appearance` / `logging` 7 section | 单一文件好管理；不分散 |
| 5 | 同步模型 = JSON 文件 + Tauri State 缓存 + emit `settings-changed` 事件 | 双窗同步；写后必发事件（CLAUDE.md 全局约束） |
| 6 | 快捷键 modifier 用 bitmask u32 + code 字符串（不用 logical modifier 抽象） | 跨平台各自存各自；v1 不做跨平台迁移 |
| 7 | UI 字段**分 basic / advanced**，advanced 默认折叠 | 用户决策负担降 75%（首屏 ~12 字段 vs 总 ~50） |
| 8 | 辅助功能 section 提到 basic（独立 section） | 当前最高优先级诊断项，浮窗不出现的直接原因之一 |
| 9 | 色散分 subtle（默认）/ vivid（v1.1 启用） | spec §八.3 极淡原则 vs 参考图营销态的张力；v1 守 spec，v1.1 加可选 |
| 10 | 日志三档：stderr + 文件（无轮转，>5MB 启动时 rename）+ 内存 ring（200） | dev 友好 + 生产可查 + 诊断页可读 |
| 11 | 日志库 `log` + `env_logger`，**不**引 `tracing` | 项目轻量；无 span/async 追踪需求 |
| 12 | 浮窗 / 主窗位置持久化在 settings.json `windowState` | 不另开文件 |
| 13 | Autostart **OS 是 source of truth**，settings.startup.autostart 是 UI 镜像 | 启动时 `autolaunch().is_enabled()` 同步一次；改 settings → 调 OS |
| 14 | 窗口位置 = last known geometry，不做"重置到屏幕中心"以外逻辑 | v1 简化 |
| 15 | macOS panel 仍走 cocoa/objc `setHidesOnDeactivate` 等公开 API | 未提交的 `macos-private-api` feature 是死配置，v1 不动（**后续 sprint 单独评估 NSVisualEffectView 真正是否需要**） |

## §2 Settings Schema

### 文件位置

```
~/Library/Application Support/com.mindtap.app/settings.json   (macOS)
%APPDATA%/com.mindtap.app/settings.json                       (Windows)
~/.config/com.mindtap.app/settings.json                       (Linux)
```

→ Tauri 2 `app.path().app_config_dir()` 拿目录，文件名固定 `settings.json`。

**写盘策略**：写到 `settings.json.tmp` → fsync → rename 覆盖（atomic）。失败保留旧文件不破坏。

### Top-level struct

```rust
// src-tauri/src/settings/schema.rs

#[derive(Debug, Clone, Serialize, Deserialize)]
#[serde(rename_all = "camelCase")]
pub struct Settings {
    pub version: u32,
    pub meta: MetaSettings,
    pub startup: StartupSettings,
    pub hotkey: HotkeySettings,
    pub floating: FloatingSettings,
    pub window_state: WindowStateSettings,
    pub appearance: AppearanceSettings,
    pub logging: LoggingSettings,
}
```

### 各 section

| Section | 字段 | 默认值 | UI 级别 |
|---|---|---|---|
| `meta` | `firstLaunchAt: i64`（unix ms；`0` 作首启动 sentinel） | `0` | 隐藏（自动） |
| | `totalLaunches: u64` | `0` | about |
| | `schemaVersion: u32` | `1` | 隐藏 |
| `startup` | `autostart: bool` | `false` | basic |
| `hotkey` | `modifiers: u32`（bitmask of `Modifiers`） | `SUPER\|SHIFT`（macOS）/ `CONTROL\|SHIFT`（其他） | basic |
| | `code: String`（`Code` 变体名字符串） | `"Space"` | basic |
| `floating` | `foldW/H: u32` | `320 / 36` | advanced |
| | `expandW/H: u32` | `360 / 280` | advanced |
| | `maxW/H: u32` | `480 / 460` | advanced |
| | `hoverOutDelayMs: u32` | `200` | advanced |
| | `animationMs: u32` | `150` | advanced |
| | `startVisible: bool` | `true` | basic |
| `windowState` | `main: Option<WindowGeometry>`（`{x, y, w, h, maximized}`） | `None` | advanced |
| | `floating: Option<WindowGeometry>`（`{x, y}`） | `None` | advanced |
| `appearance` | `theme: Theme`（`Light\|Dark\|Auto`） | `Auto` | basic |
| | `dispersion: Dispersion`（`Subtle\|Vivid`） | `Subtle` | advanced |
| `logging` | `level: LogLevel`（`Error\|Warn\|Info\|Debug\|Trace`） | `Info` | advanced |
| | `ringSize: u32` | `200` | 隐藏（hardcoded 200，字段保留） |
| | `fileEnabled: bool` | `true` | advanced |

### Hotkey 编码

**modifier 用 bitmask**（`tauri_plugin_global_shortcut::Modifiers` 是 bitflags，`SHIFT=1, CTRL=2, ALT=4, META/SUPER=8`）。`code` 用枚举变体名字符串，加载时 `Code::from_str(&code)?`。

**跨平台**：modifier 在 macOS 存 `SUPER`（=Cmd），其他平台存 `CONTROL`。**不做 logical modifier 抽象**，跨平台迁移是 v2 议题。

### Validation

```rust
impl Settings {
    pub fn validate(&self) -> Result<(), SettingsError> {
        // floating 几何：fold ≤ expand ≤ max；各值 ≥ 32px
        // animation 50~1000ms；hoverOutDelay 0~2000ms
        // ringSize 50~2000
        // logLevel 必须是已知枚举
        // hotkey.code 必须是 tauri Code::from_str 接受的字符串
    }
}
```

加载时 `parse → validate → 失败回 defaults + log::warn!`，**不 crash**。

### 迁移

```rust
pub fn load(path: &Path) -> Settings {
    let raw = match fs::read_to_string(path) {
        Ok(s) => s,
        Err(_) => return Settings::default(),
    };
    let v: serde_json::Value = serde_json::from_str(&raw).unwrap_or_default();
    match v.get("version").and_then(|x| x.as_u64()).unwrap_or(0) {
        0 => migrate_v0_to_v1(v),  // legacy → v1
        1 => serde_json::from_value(v).unwrap_or_default(),
        v if v > CURRENT_VERSION => {
            log::warn!("settings.json version {v} newer than app, ignoring");
            Settings::default()
        }
        _ => Settings::default(),
    }
}
```

`CURRENT_VERSION = 1`。v1 起步 = 0 迁移历史（greenfield）。

### Tauri command surface

```rust
#[tauri::command] pub fn settings_get(state: tauri::State<'_, SettingsState>) -> Settings
#[tauri::command] pub fn settings_set(new: Settings, app: tauri::AppHandle, state: tauri::State<'_, SettingsState>) -> Result<(), String>
#[tauri::command] pub fn settings_reset(app: tauri::AppHandle, state: tauri::State<'_, SettingsState>) -> Result<Settings, String>
```

`SettingsState(pub Mutex<Settings>)`，挂 `app.manage()`。

## §3 后端模块边界

### 初始化顺序（lib.rs setup）

```rust
.setup(move |app| {
    log::init(app.handle())?;                                          // 1
    db::init(app)?;                                                    // 2
    let s = settings::load_or_default(app.handle());                   // 3
    app.manage(SettingsState(Mutex::new(s.clone())));
    app.manage(RingState(log::ring::shared()));                        // ring 共享
    floating::ensure_window(app.handle(), &s.floating)?;               // 4
    settings::apply_hotkey(app.handle(), &s.hotkey)?;                  // 5
    settings::apply_autostart(app.handle(), s.startup.autostart)?;     // 6
    log::apply(&s.logging);                                            // 7
    for label in ["main", "floating"] { /* 菜单绑定 */ }               // 8
    Ok(())
})
```

### 4 个 side-effect appliers

| 函数 | 作用 | 调用时机 |
|---|---|---|
| `apply_hotkey(app, &hotkey)` | unregister 旧的 → register 新的 | startup + settings_set 后 |
| `apply_autostart(app, bool)` | `app.autolaunch().enable()/disable()` | startup + settings_set 后 |
| `apply_floating_geometry(app, &geom)` | `window.set_min_size/max_size` | startup + settings_set 后 |
| `apply_logging(&logging)` | `env_logger::Builder::filter_level(...)` | startup + settings_set 后（热更新） |

**apply_hotkey 失败安全模式**：先 register 新，成功后再 unregister 旧的；register 失败则保留旧快捷键并 toast。

### lib.rs 快捷键 handler 闭包

```rust
.with_handler(move |app, sc, event| {
    if event.state != ShortcutState::Pressed { return; }
    let st = app.state::<SettingsState>();
    let cur = st.0.lock().unwrap().hotkey.to_shortcut().ok();
    if Some(sc) == cur.as_ref() {
        let _ = commands::floating_cmd::floating_toggle(app.clone());
    }
})
```

→ 不再在 startup 构造常量 `toggle_shortcut`，所有快捷键来自 settings。

### settings::save atomic write

```rust
pub fn save(app: &AppHandle, s: &Settings) -> Result<(), String> {
    let dir = app.path().app_config_dir().map_err(|e| e.to_string())?;
    fs::create_dir_all(&dir).map_err(|e| e.to_string())?;
    let final_path = dir.join("settings.json");
    let tmp_path = final_path.with_extension("json.tmp");
    let json = serde_json::to_string_pretty(s).map_err(|e| e.to_string())?;
    {
        let mut f = fs::File::create(&tmp_path).map_err(|e| e.to_string())?;
        f.write_all(json.as_bytes()).map_err(|e| e.to_string())?;
        f.sync_all().map_err(|e| e.to_string())?;
    }
    fs::rename(&tmp_path, &final_path).map_err(|e| e.to_string())?;
    Ok(())
}
```

### 折叠 UI 模式

每个 section 卡片底部小一行（**per-section**，不用全局开关）：

```
┌─ 浮窗 ─────────────────────────────┐
│  启动时可见                 [▣]    │
│                                    │
│  ▸ 高级                            │  ← 点击展开
└────────────────────────────────────┘
```

实现：shadcn `Collapsible`（Glassic UI 已含）。state 不持久化（每开主窗重置为折叠）。

### KeyRecorder 状态机

```
        ┌────────┐  点"录制"  ┌──────────┐  按键   ┌────────┐
        │  idle  │ ─────────▶ │ recording│ ──────▶ │ saving │
        └────────┘            └─────┬────┘         └────┬───┘
             ▲                      │ Esc                │
             │                      ▼                    ▼
             │                  ┌────────┐          settings_set
             └──────────────────│ cancel │          → apply_hotkey
                                └────────┘          → emit changed
                                                       → idle
```

- recording 态监听 webview `keydown`，parse → `{modifiers, code}`
- 校验：必须有至少 1 个 modifier；不能只是单字母
- 实时显示：`⌘⇧Space` / `Esc 取消`
- 后端二次校验：`Code::from_str(&code)?` 失败 reject

```
  └─ tauri-bridge.api.settings_set(new)       // debounce 300ms

useAccessibility()
  └─ api.accessibility_status()               // 轮询 5s 或窗口 focus
  └─ api.accessibility_request_prompt()
  └─ api.open_ax_settings()                   // opener

useDiagnostics()
  └─ api.diagnostics_get()                    // mount + focus
  └─ api.diagnostics_recent_logs()            // LogViewer 展开
```

**不乐观更新**：所有改动走 `setLocalField → debounce → settings_set → 后端 validate/save/apply/emit → 前端订阅到 → setSettings → UI 同步`。后端 reject 时前端自然 revert。

## §5 诊断 + 日志 + 错误处理

### 日志基建

**Cargo.toml 新增**：`log = "0.4"`、`env_logger = "0.11"`。

**输出三档**：

| 输出 | 实现 | 默认 | 控制 |
|---|---|---|---|
| stderr | env_logger 默认 | ✓ | `RUST_LOG` env var |
| 文件 | `app.path().app_log_dir()/mindtap.log` | ✓ | `settings.logging.fileEnabled` |
| Ring | `Arc<Mutex<VecDeque<LogEntry>>>` 容量 200 | ✓ | 不可关（诊断页用） |

**文件策略**：单文件追加；启动时若 `mindtap.log > 5MB` → 重命名 `mindtap.log.old`（覆盖）→ 新建。每条 fsync。

**LogEntry**：

```rust
#[derive(Clone, Serialize)]
#[serde(rename_all = "camelCase")]
pub struct LogEntry {
    pub ts: i64,         // unix ms
    pub level: String,   // "error" / "warn" / "info" / "debug" / "trace"
    pub target: String,  // 模块路径
    pub message: String,
}
```

### 前端 log 转发（`src/lib/log.ts`）

```ts
export const log = {
  error: (msg: string, ...rest: unknown[]) => send("error", msg, rest),
  warn:  (msg: string, ...rest: unknown[]) => send("warn",  msg, rest),
  info:  (msg: string, ...rest: unknown[]) => isDev && send("info",  msg, rest),
  debug: (msg: string, ...rest: unknown[]) => isDev && send("debug", msg, rest),
};
```

dev 期 console 双写；Tauri command `frontend_log` append 到 ring + file（带 `[FE]` 前缀），不转发 stderr。

### Accessibility 检测（macOS）

`src-tauri/src/accessibility/mod.rs` 用 `cocoa` + `objc` 调 `AXIsProcessTrusted()` / `AXIsProcessTrustedWithOptions({prompt: true})`。非 macOS 平台 stub 返回 `true`。

**3 个 commands**：
- `accessibility_status() → bool`
- `accessibility_request_prompt()`
- `open_ax_settings()` → `x-apple.systempreferences:com.apple.preference.security?Privacy_Accessibility`

### 诊断面板数据源


**Commands**：`diagnostics_get() → Diagnostics`、`diagnostics_recent_logs() → Vec<LogEntry>`、`frontend_log(level, message, args)`。

### 错误处理边界

| 错误源 | 级别 | UI 反馈 | 日志 |
|---|---|---|---|
| `settings.json` 读/parse 失败 | WARN | 无（用 defaults） | log + ring |
| `settings_set` validate 失败 | ERROR | Toast | log + ring |
| `apply_hotkey` register 失败 | ERROR | Toast（"快捷键冲突"）；旧保留 | log + ring |
| `apply_autostart` 失败 | ERROR | Toast | log + ring |
| `apply_floating_geometry` 失败 | WARN | 无（保留旧） | log + ring |
| `save` atomic write 失败 | ERROR | Toast（"磁盘满？"）；**内存 state 保留但下次启动失效** | log + ring + 文件 |
| `accessibility_status` 系统调用失败 | WARN | "未知" badge | log |
| `db query` 失败 | ERROR | Toast + 当前页降级 | log + ring |
| 前端 `frontend_log` 自身失败 | — | 静默吞 | — |
| Rust panic | FATAL | Dialog（plugin-dialog） | log + 文件 |

**Toast**：自写 `src/lib/toast.ts`（~30 行），**不引任何 toast 库**（含 sonner）。

### 日志级别矩阵

| 模块 | 级别 |
|---|---|
| `settings::*` | INFO（成功 load/save） + ERROR（失败） |
| `floating::*` | DEBUG（show/hide） + WARN（is_visible 拿不到） |
| `commands::*` | INFO（IPC 入口） + ERROR（异常） |
| `db::*` | ERROR（query 失败） |
| `accessibility::*` | INFO（首次） + DEBUG（轮询） |
| `log::*` | ERROR（file write 失败） |

### 首启动 & metadata

```rust
fn load_or_default(app: &AppHandle) -> Settings {
    let mut s = load_from_file(...).unwrap_or_default();
    if s.meta.first_launch_at == 0 {
        s.meta.first_launch_at = now();
    }
    s.meta.total_launches += 1;
    let _ = save(app, &s);  // best-effort，失败不阻塞启动
    s
}
```

## 持久化分配（单一来源原则）

| 数据 | 落点 | Source of Truth |
|---|---|---|
| 领域数据（task/idea/check-in/record） | SQLite | DB |
| 用户偏好 | `settings.json` | JSON 文件 |
| 窗口状态 | `settings.json.windowState` | JSON 文件 |
| 元数据 | `settings.json.meta` | JSON 文件 |
| 会话状态（active task） | SQLite（`task.status='active'` + `focus_started_at`） | DB |
| 缓存 / log ring | 内存（`Mutex<VecDeque>`） | 内存 |
| Autostart | OS LaunchAgent / Registry | OS（settings.startup.autostart 是 UI 镜像） |
| Hotkey | `settings.json.hotkey` | JSON 文件（OS 不持久化全局快捷键） |
| Logging level | `settings.json.logging.level` + `RUST_LOG` env 覆盖 | JSON 文件（env 优先） |

**Mirror 原则**：mirror 不写盘。改 mirror → 改 source → mirror 刷新。
